Indien deze boodschap zichtbaar is, betekent dat dat er zich een fout heeft voorgedaan tijdens het inlezen van je student.js bestand. If this message is visible, it means that an error occurred while reading your student.js file.

  • Open dethe browserconsole.
    Windows MacOS
    Chrome Ctrl + Shift + J Cmd + Opt + J
    Firefox Ctrl + Shift + K Cmd + Opt + K
    Safari Cmd + Opt + C
  • De foutmelding zou een hint moeten geven naar wat er verkeerd gelopen is. Let ook op het regelnummer waarop de fout zich voordoet. The error message should give a hint to what went wrong. Also note the line number on which the error occurs.

Indien er geen foutmelding staat of de browser niet lijkt te reageren, betekent dit dat je vermoedelijk een oneindige lus hebt geïmplementeerd. If there is no error message or the browser does not seem to respond, it means that you have probably implemented an infinite loop.

  • Verwijder je meest recent geschreven code uit student.js. Het gemakkelijkste is door het in commentaar te zetten. Remove your most recently written code from the student.js file. The easiest is by commenting it out.
  • Refresh de pagina. Mocht dit niet lukken, sluit je de tab en heropen je ze (Ctrl + Shift + T of Command + Shift + T). Refresh the page. If this fails, close the tab and reopen it (Ctrl + Shift + T of Command + Shift + T).
  • De opgaves zouden opnieuw moeten verschenen zijn. Indien dit niet zo is, verwijder je meer code uit student.js tot alles terug werkt. The statements should have reappeared. If not, comment out more code from student.js until everything works again.
  • Herbekijk je code en tracht te ontdekken waarom je lus niet eindigt. Review your code and try to figure out why your loop doesn't stop.
